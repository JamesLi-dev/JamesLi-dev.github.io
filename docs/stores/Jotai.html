Jotai

基于 react hook，内部存储

优缺点​

优点​

  1. 使用简单
  2. 组件颗粒度较细的情况下，jotai 性能更好
  3. 支持存储全局状态

缺点​

  1. 依赖 react 上下文， 无法组件外调用，相对而言, zustand 在 react 环境外及全局可以更好地工作
  2. 受 ui 框架约束(react)

安装​


# npm
npm i jotai

# yarn
yarn add jotai

# pnpm
pnpm add jotai
